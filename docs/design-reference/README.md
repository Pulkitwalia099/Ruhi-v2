## Design System (from Stitch theme)

- **Canvas:** Warm Cream `#FFF8F5`
- **Primary:** Coral `#FF6B6B` / Deep `#AE2F34`
- **Secondary:** Teal `#006A65` / Light `#4ECDC4`
- **Tertiary:** Lavender `#655975` / Light `#A495B4`
- **Headline font:** Plus Jakarta Sans
- **Body font:** Be Vietnam Pro
- **Style:** Glassmorphic cards, editorial asymmetry, no hard borders
- **Roundness:** Fully rounded (pill shapes)

## How to Use

- Open the `.html` files in a browser to preview the designs
- Reference the color tokens and component patterns when building React components
- The Stitch project can be edited via MCP tools (`edit_screens`, `generate_variants`)
